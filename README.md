### Development

Prerequisites:

- Node.js (LTS version recommended)
- npm or yarn

```bash
# Clone the repository
git clone https://github.com/riansap/moodoro.git
cd moodoro

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Package the app
npm run package
```
